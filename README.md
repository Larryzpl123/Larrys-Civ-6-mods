领袖能力 - 铁血政策

美国的新领袖 - 托马斯·杰斐逊 (制作中)   
领袖能力 - 人民之友   
城市宜居度 +1。购买地块成本降低 18.03%。每个专业化区域 +1 金币，大学 +1 科技。   

印加的新领袖 - 图帕克·印卡·尤潘基 (制作中)   
领袖能力 - 点石成金   
山脉和有资源的单元格 +1 金币，货币后 +2，银行后 +3，经济后 +4。   

美国的新领袖 - 安德鲁·杰克逊 (制作中)   
领袖能力 - 老胡桃木   
单位受伤不遭受战斗力减益，受伤的单位+3战斗力。种植园+1金币，银行+3金币。   
特色单位 - 民兵   
安德鲁·杰克逊作为领袖时的特色近战兵种，代替线列步兵。战斗力与线列步兵相同，生产消耗略低（基准消耗-300生产力），与战斗力更高的单位战斗时+3战斗力。

# 新领袖与新文明

历史文明

纳巴泰帝国 - 哈里萨四世   
领袖能力 - 佩特拉之权   
在首都6单元格范围内（不包括首都）建立城市时，获得一个免费建造者。所有6单元格外的非首都城市的所有产出 -50%。   
文明能力 - 果树农业   
沙漠地块（非泛滥平原）提供 +2 食物、+2 金币和 +1 生产力。   

阿尤布 - 萨拉丁 (制作中)   
领袖能力 - 信仰之义   
所有区域和建筑+1信仰（包括市中心与市中心建筑），重骑兵+1战斗力，可以用信仰购买重骑兵。   
阿尤布王朝（埃及、阿拉伯、美索不达米亚）   
文明能力 - 繁荣一统   
开局获得占星术尤里卡，解锁占星术后首都单元格+1全产出。   
圣地及其建筑+1金币，军营及其建筑+1科技，剧院及其建筑+1生产力。   
特色单位 - 世袭贵族战车射手   
同埃及的世袭贵族战车射手
特色单位 - 奴隶兵   
萨拉丁（信仰之义）作为领袖的特色单位。同阿拉伯的奴隶兵   
特色改良 — 狮身人面像   
萨拉丁（信仰之义）作为领袖的特色改良设施。同埃及的狮身人面像。   
特色改良 — 金字形神塔   
同苏美尔的金字形神塔。   

苏联 - 列宁 (制作中)    
领袖能力 - 众志成城   
拥总督的城市中每位公民将提供 0.6 点生产力。+16% 生产力。   
文明能力 - 工农联合   
市中心单元格 +1 生产力。矿山 +1 食物, +1 金币，建造完特色工业区赠送一个建造者。   
特色工业区 - 重工业区   
苏联的特色工业区。半价。矿山提供标准相领加成，学院和水渠提供大量相邻加成。别的所有方式都不提供相邻加成（包括区域的少量相邻加成）。   
特色单位 - 红军   
代替步兵。战斗力相同。生产力7折。每个相领红军 +1 力。
